## 自动分享链接至指定网站

这个尝试，是想基于 `Puppeteer` 抓取某链接（ 具体是在 https://jeffjade.com/categories/Front-End/ 中随机出一篇），将其推送到 [技术头条](http://blogread.cn/news/)。

### 运行命令
```
git clone https://github.com/nicejade/toss-puppeteer
npm i (yarn)
npm run shareBlogToBlogread
```
